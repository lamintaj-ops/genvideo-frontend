# Aquaverse Video Generator - Frontend

Static frontend for Aquaverse Video Generator.

## Setup

1. Replace YOUR-RENDER-URL in index.html with your actual Render backend URL
2. Enable GitHub Pages: Settings  Pages  Source: main branch

## Features

-  Beautiful UI for video generation
-  Responsive design
-  Fast loading with GitHub Pages CDN
-  Multiple theme presets

## Backend

Backend API runs separately at: https://github.com/lamintaj-ops/genvideo

## Deploy

This repo is automatically deployed to GitHub Pages at:
https://lamintaj-ops.github.io/genvideo-frontend/
